# ME 模組儲存匯流排

ME 模組儲存匯流排是能以模組名稱，或是模組 ID 篩選的 <ItemLink id="ae2:storage_bus" />。

如果你想同時篩選多個模組，請使用逗號「,」分隔多個模組 ID。
